# CS222_S26_P2_G1

Story 1 implements a system that extracts all available course numbers from multiple files (courses.txt, courses1.txt, courses2.txt).

The program reads course data from files, extracts the course number from each line, removes duplicate entries using a set, and returns a sorted list of unique course numbers.

This design ensures that only distinct course offerings are displayed, even if the same course appears multiple times across different files. The implementation follows clean code principles by separating file reading, data extraction, and processing logic into independent functions.
